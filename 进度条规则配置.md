# 进度条规则配置文档

## 项目概述
本文档用于定义比特币自主托管网站中进度条的显示规则。用户可以选择硬件签名器、软件钱包和区块链节点的不同组合，每种组合需要对应不同的进度条百分比。

## 当前系统行为
- **硬件签名器**：单选，包括"不使用签名器"选项。
- **软件钱包**：单选
- **区块链节点**：单选

## 所有可能的用户选择情况

### 第一类：选择"不使用硬件签名器"的情况

#### 1. 仅选择"不使用签名器"
- **用户选择**：仅勾选"不使用签名器"
- **进度条百分比**：0%
- **备注**：

#### 2. "不使用签名器" + 软件钱包
- **用户选择**："不使用签名器" + 任意一个软件钱包
- **进度条百分比**：60%
- **备注**：

#### 3. "不使用签名器" + 软件钱包 + 区块链节点
- **用户选择**："不使用签名器" + 软件钱包 + 区块链节点
- **进度条百分比**：80%
- **备注**：

### 第二类：选择硬件签名器的情况

#### 4. 选择硬件签名器
- **用户选择**：选择一个硬件签名器（不包括"不使用签名器"）
- **进度条百分比**：50%
- **备注**：

#### 6. 硬件签名器 + 软件钱包
- **用户选择**：硬件签名器 + 软件钱包
- **进度条百分比**：100%
- **备注**：

#### 7. 硬件签名器 + 软件钱包 + 区块链节点
- **用户选择**：硬件签名器 + 软件钱包 + 区块链节点
- **进度条百分比**：120%
- **备注**：


### 第三类：空状态

#### 11. 什么都没选择
- **用户选择**：未选择任何组件
- **进度条百分比**：0%
- **备注**：


## 特殊效果配置

### 庆祝动画
- **触发条件**：进度条达到 120%
- **动画效果**：🎉 emoji闪烁3秒
- **备注**：

### 延伸效果
- **触发条件**：进度条达到 100%
- **效果描述**：进度条右侧延伸灰色区域
- **备注**：

### 溢出效果
- **触发条件**：进度条超过 120%
- **效果描述**：进度条自然溢出，颜色渐变
- **备注**：



## 实现注意事项

### 技术要求
- 进度条需要平滑动画过渡（0.5秒）
- 百分比文字需要居中显示
- 支持响应式设计

### 用户体验要求
- 进度变化要有逻辑性
- 给用户明确的完成感
- 鼓励用户选择最佳实践
